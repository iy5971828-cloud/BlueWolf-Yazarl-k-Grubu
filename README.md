# BlueWolf Yazarlık Grubu

GitHub Pages: https://bluewolf.github.io
License: MIT

BlueWolf Yazarlık Grubu'nun resmi web sitesi. Edebiyat, şiir, öykü ve yaratıcı yazım alanında çalışmalar yapan bir yazarlık topluluğu.

## Canlı Web Sitesi
Web sitemiz şu adreste yayında: https://bluewolf.github.io

## Proje Hakkında
Bu depo, BlueWolf Yazarlık Grubu'nun resmi web sitesinin kaynak kodlarını içermektedir. Site tamamen statik HTML, CSS ve JavaScript kullanılarak oluşturulmuştur.

## Özellikler
- Responsive ve mobil uyumlu tasarım
- Grup üyeleri için profil sayfaları
- Eserler bölümü (şiir, öykü, denemeler)
- Etkinlik takvimi ve duyurular
- İletişim formu
- Sosyal medya entegrasyonu

## Teknolojiler
- HTML5
- CSS3
- JavaScript (vanilla)
- GitHub Pages (hosting)
